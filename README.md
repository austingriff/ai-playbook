# Claude Playbook Bootstrapper

Clone this repo, open it in Claude Code, and generate a custom Claude strategy for your organization in under an hour.

## What It Does

Runs you through a short questionnaire about your team and stack, then uses Claude to generate a tailored configuration strategy — ready to deploy across your repos.

## Quick Start

```bash
git clone <this-repo> my-claude-playbook
cd my-claude-playbook
claude
```

Once inside Claude Code:

```
/setup
/generate-playbook
/sync-org
```

## Requirements

- [Claude Code](https://claude.ai/code) installed and authenticated
- Git
